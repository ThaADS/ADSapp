## Color Scheme

### Primary Colors
- **Blue-600:** `#2563EB` - Primary buttons, active states
- **Blue-50:** `#EFF6FF` - Background gradient
- **White:** `#FFFFFF` - Form background

### Status Colors
- **Green-600:** `#16A34A` - Success states, checkmarks
- **Red-600:** `#DC2626` - Error states, validation errors
- **Yellow-600:** `#CA8A04` - Warning states
- **Gray-600:** `#4B5563` - Inactive states, helper text

### Text Colors
- **Gray-900:** `#111827` - Headings
- **Gray-700:** `#374151` - Labels
- **Gray-600:** `#4B5563` - Body text
- **Gray-500:** `#6B7280` - Helper text

### Keyboard Navigation

### Mobile First (375px - 767px)
- Single column layout
- Full-width inputs
- Stacked buttons
- Compact progress indicator

### Tablet (768px - 1023px)
- Centered layout (max-width: 768px)
- Larger form fields
- Side-by-side buttons

### Desktop (1024px+)
- Centered layout (max-width: 900px)
- Optimal spacing
- Full progress indicator with labels
