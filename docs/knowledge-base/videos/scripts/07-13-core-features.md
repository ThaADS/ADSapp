# Video 7: Message Templates - Create and Use Professional Templates

**Duration**: 5-7 minutes
**Learning Objectives**: Master template creation, understand approval process, use templates effectively

## Script Outline

**Introduction (0:00-0:30)**: Templates ensure consistency and save time. Two types: Quick Replies and WhatsApp Templates.

**Section 1: Quick Replies (0:30-2:00)**: Create instant responses for common questions. Demo creating quick reply with shortcut. Show usage with / command.

**Section 2: WhatsApp Templates (2:00-4:00)**: Create templates for customer-initiated messages. Explain approval process. Show variable placeholders. Demonstrate submission to WhatsApp.

**Section 3: Template Best Practices (4:00-5:00)**: Keep messages conversational. Use variables for personalization. Organize by category. Test before widespread use.

**Conclusion (5:00-5:30)**: Templates created, approval understood, ready to streamline communication.
